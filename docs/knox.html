Samsung Knox

Samsung Knox was a unique project in that it targeted two completely different audiences on one url. I was able to act as lead developer on the project, establishing architecture for the page, implementing pixel perfect styles which are fully responsive, and making all interactive elements on the page from scratch.

View it Live

Interactive Elements

A main feature of the page was an interactive phone which expands to give the user more details about each layer in the security platform. This section was built entirely in css, with click events only setting which layer was active. Using sass, a dynamic system for structuring each layer to line up perfectly with the intro video was integral as the component was in development before we had the final creative to match it against.
