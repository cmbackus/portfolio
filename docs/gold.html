Godiva Gold

The Godiva Gold hero features a chocolate box where the user can discover each unique chocolate in the collection. The experience opens with a shimmering box on display, activated by mouse movement or accelerometer on mobile. The box then opens, and clicking each piece reveals additional details and content. The project was done with javascript and css animations.

View it Live

Interactive Elements

Each chocolate is clickable to reveal a content box complete with movie player. In order to make this hero fully responsive, the position of pop-ups depends on screen width, and changes to a carousel on mobile. This project used scene7 image and video assets, and demandware for content management on the back-end.
